# Usage

<!-- usage -->

```sh-session
$ npm install -g get-latest-version
$ get-latest-version COMMAND

$ get-latest-version (-v|--version|version)
get-latest-version/0.0.0 win32-x64 node-v10.16.2
$ get-latest-version --help [COMMAND]
USAGE
  $ get-latest-version COMMAND
...
```
